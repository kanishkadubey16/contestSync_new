# 🚀 ContestSync — All Competitive Programming Contests in One Place

ContestSync helps competitive programmers stay updated by aggregating upcoming contests from multiple platforms like **Codeforces**, **LeetCode**, **AtCoder**, **HackerRank**, and more!

---

## ⭐ Features

- 🔄 **Contest Aggregation** from multiple platforms via APIs  
- 🔐 **User Authentication** using JWT  
- 🔍 **Filter & Search Contests** by platform & timing  
- ⏰ **Set Contest Reminders** (Email/Notifications)  
- 📊 **Contest History Tracking** with charts  
- 🛠️ **Admin Dashboard** for managing users & system analytics  
- 🏆 **Leaderboard** based on participation  
- 🌙 **Dark/Light Mode** UI Theme  
- ⏱️ **Auto-Sync Cron Jobs** to refresh contests regularly  

---

## 🧱 Tech Stack

### Frontend
- ⚛️ React.js + React Router
- 🎨 Tailwind CSS
- 📈 Chart.js
- 🔗 Axios

### Backend
- 🟢 Node.js + Express.js
- 🗄️ Prisma ORM + PostgreSQL
- 🔑 JWT Authentication
- 🕒 Node-cron
